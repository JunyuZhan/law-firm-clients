## 🎯 系统简介

客户服务系统是律所管理系统的一部分，为客户提供便捷的项目查询和文件管理服务。

### 主要功能

- 📋 **项目查询**：查看项目基本信息、进度和状态
- 📁 **文件管理**：上传、下载、查看项目相关文件
- 📧 **消息通知**：接收项目更新通知（短信、微信、邮件）

---

## 🚀 快速开始

### 访问系统

1. **获取访问链接**

   律所会通过短信、微信或邮件向您发送项目访问链接，格式如下：
   ```
   http://your-domain.com/portal/matter/项目ID?token=访问令牌
   ```

2. **打开链接**

   点击链接或复制到浏览器地址栏打开，系统会自动跳转到项目详情页面。

3. **查看项目信息**

   在项目详情页面，您可以查看：
   - 项目基本信息（名称、编号、状态等）
   - 项目进度和更新
   - 相关文件列表

## 📖 功能说明

### 1. 项目详情查看

#### 项目基本信息

在项目详情页面顶部，您可以查看：

- **项目名称**：项目的名称
- **项目编号**：项目的唯一编号
- **客户名称**：您的姓名或公司名称
- **项目状态**：当前项目状态（进行中、已完成等）
- **有效期至**：项目访问的有效期限
- **创建时间**：项目创建的时间

#### 项目进度

项目进度信息显示在项目详情中，包括：

- 当前阶段
- 已完成事项
- 待办事项
- 重要时间节点

### 2. 文件管理

#### 上传文件

1. 在项目详情页面，找到"文件管理"区域
2. 点击"上传文件"按钮
3. 在弹出的对话框中：
   - 点击"选择文件"按钮，选择要上传的文件
   - 选择文件分类（合同文件、证据材料、法律文书、其他）
   - 输入文件说明（可选）
4. 点击"确定"按钮完成上传

**注意事项**：
- 单个文件大小不超过10MB
- 支持常见文件格式（PDF、Word、Excel、图片等）
- 文件上传后无法修改，请确认无误后再上传

#### 查看文件列表

在"文件管理"区域，您可以查看：

- 所有已上传的文件
- 文件名称、大小、分类
- 上传时间

#### 下载文件

1. 在文件列表中，点击文件名
2. 文件会自动下载到您的设备

#### 删除文件

1. 在文件列表中，找到要删除的文件
2. 点击"删除"按钮
3. 确认删除操作

**注意事项**：
- 删除操作不可恢复，请谨慎操作
- 只能删除自己上传的文件

### 3. 消息通知

系统会通过以下方式向您发送项目更新通知：

- **短信**：发送到您预留的手机号码
- **微信**：发送到您绑定的微信公众号
- **邮件**：发送到您预留的邮箱地址

通知内容包括：
- 项目状态更新
- 重要时间节点提醒
- 文件上传通知
- 其他重要信息

## ❓ 常见问题

### Q1: 无法打开访问链接？

**A**: 请检查：
1. 链接是否完整（包含项目ID和token）
2. 链接是否已过期（检查有效期）
3. 网络连接是否正常

如果问题仍然存在，请联系律所工作人员。

### Q2: 文件上传失败？

**A**: 可能的原因：
1. 文件大小超过10MB限制
2. 文件格式不支持
3. 网络连接不稳定

**解决方案**：
- 压缩文件或分割大文件
- 检查文件格式是否支持
- 检查网络连接后重试

### Q3: 看不到项目信息？

**A**: 可能的原因：
1. 项目访问已过期
2. 项目状态已变更
3. 浏览器缓存问题

**解决方案**：
- 联系律所获取新的访问链接
- 清除浏览器缓存后重试
- 使用无痕模式打开链接

### Q4: 如何修改上传的文件？

**A**: 系统不支持修改已上传的文件。如需修改：
1. 删除原文件
2. 重新上传修改后的文件

### Q5: 忘记访问链接怎么办？

**A**: 请联系律所工作人员，他们会重新发送访问链接给您。

### Q6: 可以同时查看多个项目吗？

**A**: 可以。每个项目都有独立的访问链接，您可以保存多个链接，分别访问不同的项目。

### Q7: 文件下载后在哪里？

**A**: 文件会下载到您设备的默认下载目录：
- **Windows**：通常在"下载"文件夹
- **Mac**：通常在"下载"文件夹
- **手机**：通常在"文件"应用的"下载"文件夹

### Q8: 系统支持哪些浏览器？

**A**: 推荐使用以下浏览器：
- Chrome（最新版本）
- Firefox（最新版本）
- Safari（最新版本）
- Edge（最新版本）
