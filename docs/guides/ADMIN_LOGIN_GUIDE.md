# 客服系统管理员登录指南

> 文档版本：v2.0  
> 创建日期：2026-02-02  
> 更新日期：2026-02-02

## 📋 概述

客服系统管理后台使用**用户名密码 + JWT Token**认证机制。管理员需要使用用户名和密码登录，登录成功后系统会返回JWT Token，后续访问管理后台API时需要在请求头中携带此Token。

**重要说明**：
- **管理员登录**：使用用户名密码 + JWT Token（本文档说明）
- **系统对接**：使用API密钥认证（用于律所系统与客服系统的数据对接）

```
http://localhost:3000/admin/login
```

### 步骤2：输入用户名和密码

登录成功后，您可以访问以下管理后台功能：

- **API密钥管理**：`/admin/api-keys`（用于系统对接）
- **通知模板管理**：`/admin/notification-templates`
- **系统配置**：`/admin/config`

## 🔒 认证机制说明

### JWT Token认证流程

```
1. 用户访问管理后台页面
   ↓
2. 路由守卫检查是否已登录（检查localStorage中的Token）
   ↓
3. 如果未登录 → 跳转到 /admin/login
   ↓
4. 用户输入用户名密码 → 调用登录API
   ↓
5. 后端验证用户名密码 → 生成JWT Token
   ↓
6. 前端保存Token到localStorage
   ↓
7. 访问管理后台API时 → 自动在请求头添加 Authorization: Bearer {token}
   ↓
8. JWT过滤器验证Token → 验证通过后返回数据
```

### Token存储位置

- **前端存储**：浏览器的 `localStorage`（键名：`admin_token`）
- **用户信息**：浏览器的 `localStorage`（键名：`admin_user`）

### Token验证

每次调用管理后台API时：
1. 前端自动在请求头添加：`Authorization: Bearer {token}`
2. JWT过滤器验证Token：
   - 检查Token格式是否正确
   - 检查Token是否过期
   - 检查用户是否存在且已启用
3. 验证通过后返回数据，失败返回401错误并跳转到登录页

### Token有效期

- **默认有效期**：24小时
- **配置位置**：`application.yml` → `client-service.jwt.expiration-hours`
- **过期后**：需要重新登录

## 🔐 账户安全

### 账户锁定机制

为了防止暴力破解，系统实现了账户锁定机制：

- **失败次数限制**：连续登录失败5次后，账户会被锁定
- **锁定时间**：30分钟
- **锁定后**：无法登录，需要等待锁定时间到期或联系管理员解锁

### 密码强度要求

修改密码时，新密码必须满足以下要求：

- **最小长度**：8位
- **必须包含**：至少一个字母和一个数字
- **示例**：`admin123`、`password123`、`Admin123`

### 登录日志

系统会记录所有登录尝试：

- **成功登录**：记录用户名、IP地址、User-Agent、登录时间
- **失败登录**：记录用户名、IP地址、User-Agent、失败原因
- **查看位置**：数据库 `admin_login_log` 表

## ⚠️ 常见问题

### Q1: 忘记密码怎么办？

**A**: 
1. 联系系统管理员重置密码
2. 或通过数据库直接修改密码（需要BCrypt加密）

### Q2: 账户被锁定怎么办？

**A**: 
1. 等待30分钟后自动解锁
2. 或联系系统管理员手动解锁（清除 `failed_login_count` 和 `locked_until` 字段）

### Q3: Token过期怎么办？

**A**: 
1. 系统会自动检测Token过期
2. 过期后会自动跳转到登录页
3. 重新登录即可获取新的Token

### Q4: 如何退出登录？

**A**: 
- 在管理后台右上角点击用户名下拉菜单
- 选择"退出登录"
- 或手动清除浏览器localStorage中的 `admin_token` 和 `admin_user`

### Q5: 可以同时登录多个设备吗？

**A**: 
- 可以，每个设备登录后会生成独立的Token
- Token之间互不影响
- 退出登录只影响当前设备

### Q6: API密钥和管理员登录有什么区别？

**A**: 
- **管理员登录（JWT Token）**：用于管理员访问管理后台，进行系统维护和配置
- **API密钥**：用于律所系统与客服系统的数据对接（系统到系统）
