## 📝 配置建议

### 推荐配置方式

```env
# 系统完整名称（用于版权、页面标题等）
VITE_APP_NAME=贵州威迪律师事务所客户服务系统

# 律师事务所名称（用于首页显示）
VITE_LAW_FIRM_NAME=贵州威迪律师事务所

# 门户标题（可选，不配置则使用 APP_NAME）
# VITE_PORTAL_TITLE=客户服务门户

# 系统标语（首页大标题）
VITE_APP_SLOGAN=专业 · 诚信 · 高效

# 系统简称（管理后台侧边栏）
VITE_APP_SHORT_NAME=贵州威迪律师事务所客服系统

# 系统英文名称（蓝色背景条副标题）
VITE_APP_NAME_EN=Guizhou Weidi Law Firm Client Service System

# 系统英文简称（管理后台侧边栏副标题）
VITE_APP_SHORT_NAME_EN=Weidi Law Firm Service
```
